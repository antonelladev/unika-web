# 👗 Unika Boutique - Sitio Web Oficial

¡Bienvenido al repositorio de **Unika Vestidos**! Este es un proyecto de desarrollo y diseño web enfocado en la alta costura a medida para novias, damas de honor y vestidos de gala. El sitio está completamente optimizado y desplegado en producción.

🌐 **Sitio en vivo:** [unika.com.bo](https://unika.com.bo)

---

## 🚀 Características del Proyecto

* **Diseño Elegante y Minimalista:** Interfaz de usuario (UI) limpia, adaptada a la estética de la moda nupcial y de gala.
* **Totalmente Responsivo:** Adaptado a dispositivos móviles, tablets y ordenadores de escritorio.
* **Secciones Principales:** Inicio, Contacto, Portafolio de Diseños y Blog de Tendencias.

---

## 🛠️ Tecnologías y Herramientas Utilizadas

* **CMS:** WordPress
* **Maquetador Visual:** Elementor Pro
* **Estilos y Personalización:** HTML5 / CSS3 personalizado para tipografías y espaciados dinámicos.
* **Despliegue:** Servidor de Hosting independiente con configuración de dominio regional `.com.bo`.

---

## 📁 Estructura del Repositorio

Este repositorio contiene los recursos clave del desarrollo, incluyendo:
* `/assets` - Capturas de pantalla y elementos de diseño.
* `/custom-css` - Fragmentos de código CSS añadidos para mejorar la interfaz (como los efectos de texto y espaciados del banner principal).
* `README.md` - Documentación general del proyecto.
